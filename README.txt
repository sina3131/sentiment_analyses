Sentiment Analysis of Amazon Reviews
This code contains an analysis of the Amazon Reviews dataset using the NLTK SentimentIntensityAnalyzer and pandas. The goal of this analysis is to perform sentiment analysis on the reviews and extract sentiment scores for further analysis.

Dataset
The dataset used for this analysis is the Amazon Reviews dataset obtained from Kaggle. It contains a collection of reviews for various products sold on Amazon. The dataset includes information such as the review text, product rating, and other metadata.
Download the dataset from this link:
https://www.kaggle.com/datasets/bittlingmayer/amazonreviews


Analysis Steps
Sentiment Analysis: The NLTK SentimentIntensityAnalyzer was employed to calculate sentiment scores for each review in the dataset. The SentimentIntensityAnalyzer provides a compound sentiment score that represents the overall sentiment polarity of a text.

Results and Insights: The sentiment scores were extracted and analyzed to gain insights into the sentiment distribution of the reviews. Visualizations, statistical summaries, or any other relevant analysis techniques were employed to better understand the sentiment patterns within the dataset.

Requirements
The following dependencies are required to reproduce the analysis:

Python 3.x
pandas
NLTK (Natural Language Toolkit)
Usage
To replicate the analysis:

Clone this repository to your local machine.
Install the required dependencies using pip install -r requirements.txt.
Place the Amazon Reviews dataset (amazon_reviews.csv) in the data/ folder.
Run the analysis.ipynb Jupyter Notebook to execute the analysis code and generate the results.
Results
The results of the sentiment analysis are available in the Jupyter Notebook analysis.ipynb. The notebook includes visualizations, summary statistics, and any additional analysis conducted on the sentiment scores.
